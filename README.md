# How to use

- Put `AntiCheat.luau` in a `Module Script`
- Put `callAntiCheat.luau` in a `Script` 
- You can customize your configuration based on `antiCheatCall.luau` `config` variable

And done! Your anti cheat will be up and working!

## FEATURES

- Fly detection;
- No clip detection;
- Air jump detection;
- Speed hack detection.
